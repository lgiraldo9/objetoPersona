Ejercicio: Uso de Clases, Constructores, atributos, getters y setters y
Arreglos (Array)
Desarrolle un programa en Java utilizando programación orientada a objetos y arreglos
(array).
Requisitos del ejercicio
1. Crear una clase llamada #NombredelEstudiante que tenga los siguientes
atributos:
o id
o cedula
o nombre
o apellido
o edad
o sueldo
2. Crear una clase principal donde se realice lo siguiente:
o Crear un arreglo (array) de 6 personas.
o Instanciar y almacenar 6 objetos de tipo #NombredelEstudiante dentro
del arreglo.
3. El programa debe:
o Mostrar en pantalla los datos de la persona número 5 del arreglo.
o Mostrar en pantalla el total de personas almacenadas en el arreglo con el
metrodo foreach.
Indicaciones
• Utilice constructores para inicializar los datos de cada persona.
• Cree un método para mostrar la información de la persona.
• El programa debe ejecutarse desde el método main.
Entrega
El estudiante debe entregar:
• El programa debe Entregar pantallazos del proceso de todo el proceso
